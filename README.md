### 🚀 Things I've built

#### 🎮 Gemax — multiplayer 3D voxel world
A browser-first multiplayer voxel sandbox with real-time combat, six tiers of monsters, procedurally generated biomes, cities, caves and portals — no download required. Built with a TypeScript monorepo (React + PlayCanvas on the client, Node + Socket.io on the server, PostgreSQL for persistence).

#### 🔐 [react-nice-input-password](https://github.com/renanborgez/react-nice-input-password) · ⭐ 10
A React password input with a built-in, configurable strength meter powered by regex rules. Material UI compatible.

#### 🎭 [ohmyzsh-theme-renanborgez](https://github.com/renanborgez/ohmyzsh-theme-renanborgez)
